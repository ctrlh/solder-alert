# Solder Alert Enclosure

Developed using [OpenSCAD](https://openscad.org).

I recommend printing using transparent or white PETG, which is a bit more flexible. PLA is a second choice.

Designed specifically to fit the 1/4-size Adafruit breadboard and Waveshare 1.47 inch display (ESP32-S3 or ESP32-C6 variants).

It's a tight fit with about 0.5mm of the display proud of the front surface. Angle the USB connector into it's exit slot first, then firmly push down on the other end of the board to complete the insertion.

The cavity for the USB-C connector will fit common USB-whatever-to-USB-C adapters, in addition to the usual USB-C male cables.
